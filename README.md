# ZéFish – Sistema para Pesqueiro

   Projeto desenvolvido como Trabalho de Conclusão de Curso (TCC) no curso de Análise e Desenvolvimento de Sistemas, com o objetivo de modernizar e otimizar a gestão operacional de pesqueiros por meio de uma aplicação web completa e responsiva.

   Sobre o Projeto:
   O ZéFish é um sistema web desenvolvido para solucionar a baixa digitalização presente no setor de pesqueiros, oferecendo uma plataforma moderna, funcional e acessível tanto para clientes quanto para administradores.
A aplicação foi construída com foco em:
 - Experiência do usuário (UX)
 - Responsividade (mobile-first)
 - Organização e automação de processos internos
 - Facilidade de gerenciamento
A análise de mercado evidenciou que muitos pesqueiros utilizam sistemas desatualizados ou pouco estruturados, o que motivou o desenvolvimento de uma solução mais robusta e eficiente.

   Principais Funcionalidades
   -> Usuário (Cliente)
      - Cadastro e autenticação de usuários
      - Recuperação de senha
      - Visualização de eventos
      - Visualização de cardápio
      - Agendamento de pesca com calendário
      - Pagamento online via PIX
      - Recebimento de notificações
   
   -> Administrador
      - Gerenciamento de usuários
      - Controle de reservas
      - Gestão de eventos
      - Administração do cardápio
      - Controle de tanques (espécies, quantidade, capacidade...)
      - Visualização de relatórios
